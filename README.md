# Vivasayi AI Super App

Enterprise-grade mobile and web foundation for the Vivasayi AI Super App.

## Overview

This repository contains the scaffold for a cross-platform Flutter mobile app, a PHP 8.3 REST API backend, and a Bootstrap 5 admin panel. The initial foundation includes authentication, environment configuration, database schema, and architecture documentation.

## Contents

- `flutter_app/` — Flutter mobile app scaffold with clean architecture and core services.
- `php_api/` — PHP REST API backend scaffold using modern structure and JWT authentication.
- `admin_panel/` — Bootstrap 5 admin UI scaffold with responsive dashboard shell.
- `database/` — MySQL schema and seed scripts.
- `docs/` — Architecture, coding standards, naming conventions, and dependency lists.
- `infra/` — Environment examples and deployment configuration.

## Goals for Phase 1

- Set up Git-ready structure
- Enable JWT authentication and OTP login support
- Define role-based access control
- Support offline-first design
- Establish security best practices
- Prepare for Tamil and English localization
- Use Material 3 design principles in mobile app
