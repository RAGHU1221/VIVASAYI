# Dependencies and Tooling

## Flutter Mobile

- Flutter SDK 3.x or newer
- `flutter_localizations`
- `go_router`
- `dio`
- `flutter_secure_storage`
- `json_serializable`
- `provider` or `riverpod`
- `shared_preferences`
- `connectivity_plus`
- `flutter_svg`

## PHP Backend

- PHP 8.3
- `ext-pdo`
- `ext-json`
- `firebase/php-jwt`
- `vlucas/phpdotenv`
- `nikic/fast-route`
- `symfony/http-foundation`
- `monolog/monolog`

## Admin Panel

- Bootstrap 5
- Font Awesome 6
- Vanilla JavaScript

## Database

- MySQL 8.0+
- InnoDB storage engine

## DevOps / Infrastructure

- Git
- Docker (optional)
- CI/CD pipeline with GitHub Actions or GitLab CI
- SSL/TLS for API and admin panel
- Logging and monitoring tools
