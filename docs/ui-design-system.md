# UI Design System

## Design Principles

- Mobile-first with responsive behavior for tablet
- Material 3 design language
- Light and dark themes
- Accessible color contrast and typography
- Clean layouts with whitespace and glassmorphism accents
- Reusable components with consistent spacing and elevation

## Color Palette

- Primary: #2F7A2E
- Primary Container: #D7F5D4
- Secondary: #5A9E58
- Background Light: #F6FBF6
- Background Dark: #081409
- Surface Light: #FFFFFF
- Surface Dark: #112012
- Error: #B00020
- Success: #1F7D31
- Warning: #FFB020
- On Primary: #FFFFFF
- On Background: #0B2B0C

## Typography

- Display / Headings: Inter, 700
- Body text: Inter, 400
- Button / Label: Inter, 600
- Small captions: Inter, 500

## Icons

- Use Material Icons for core navigation
- Use custom agricultural glyphs for farm, crop, weather, AI, and analytics
- Maintain icon size consistency across navigation and cards

## Components

### Buttons

- Primary button
- Secondary button
- Tertiary text button
- Icon button
- Elevated button with shadow and rounded corners

### Cards

- Dashboard summary card
- Metric analytics card
- Action card with CTA
- List item card with avatar
- Glassmorphism statistic card

### Input Fields

- Filled text fields
- Outlined dropdowns
- Search field with pre-filled icon
- OTP input boxes
- Date picker / calendar field

### Navigation

- Bottom navigation with 4 items
- Drawer menu for extended navigation
- Top app bar with search and language toggle

### Widgets

- Progress indicator
- Skeleton loader cards
- Empty state illustrations
- Error page with retry action
- Animated onboarding screens
